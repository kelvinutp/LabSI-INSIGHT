Conexión WiFi

IMPORTANTE

Si es la primera vez que va a configurar la red WiFi, debe conectarse por SSH utilizando cable Ethernet, en la sección >>[Primera configuración]<<se detalla paso a paso cómo realizarla correctamente.

Paso 1

En la consola escribir el siguiente comando

sudo nano /etc/wpa_supplicant/wpa_supplicant.conf

Paso 2

Se nos abrirá la siguiente ventana

Debemos escribir los siguiente

network={
	ssid="Nombre_la_red_WiFi"
	psk="Contraseña_de_la_red_WiFi"
	key_mgmt=WPA-PSK
}

IMPORTANTE

El nombre de la red WiFi y la contraseña deben estar correctamente escritas

Paso 3

Presionar las siguientes teclas en el siguiente orden para guardar los cambios.

  • Ctrl x
  • s (para versiones en español)
  • y (para versiones en ingles)
  • Enter

Paso 4

Ya tenemos nuestra red registrada a continuación, apagaremos nuestra Raspberry desde la consola con el siguiente comando

sudo shutdown now

NOTA

En este punto podemos desconectar nuestra Raspberry de nuestra PC, si todo está correctamente configurado podemos prender nuestra raspberry y conectarnos sin necesidad del cable Ethernet

Conexión SSH Wifi

Para la conexión SSH por Wifi necesitamos conocer la ip que está asociada a nuestra raspberry, este valor lo podemos obtener por medio de diferentes aplicaciones, a continuación, se recomienda algunos software y apps que nos pueden ser de utilidad para monitorear la conexión de nuestro raspberry.

Advance IP scaner (Windows)

Fing (Android)

Paso 1

Este paso será realizado con los programas recomendados

Advance ip scaner

Damos click en Sacn

Como podemos ver la ip asociada a nuesta raspberry es la 192.168.1.28

Fing

Abrimos la aplicación Fing y tocamos la opción Online device

Hacemos Refesh

Como podemos ver desde el móvil también podemos obtener la ip asociada a nuestra raspberry

en este caso su ip asociada es la 192.168.1.28

Paso 2

Abrimos PuTTY y colocamos la ip obtenida en el Host name, el Port es el 22, posterior damos click en ”Open”

Esto nos lanzara un mensaje, damos Click en accept

Iniciamos sesión

login as: pi

pi@raspberrypi.local's password: raspberry

IMPORTANTE: La contraseña no se muestra al escribirla, solo debemos escribirla correctamente y dar enter.
