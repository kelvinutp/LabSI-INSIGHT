Programas necesarios

Para la correcta conexión entre el PC y la tarjeta embebida RaspberryPi necesitamos instalar el software PuTTY y FileZilla.

PuTTY ( Download PuTTY - a free SSH and telnet client for Windows )

PuTTY es un cliente SSH, Telnet, rlogin, y TCP raw con licencia libre. Disponible originalmente solo para Windows, ahora también está disponible en varias plataformas Unix, y se está desarrollando la versión para Mac OS clásico y Mac OS X.

FileZilla ( FileZilla - The free FTP solution (filezilla-project.org) )

FileZilla es una aplicación FTP libre y de código abierto que consta de un cliente y un servidor. Soporta los protocolos FTP, SFTP y FTP sobre SSL/TLS.

Conexión ssh

Paso 1: Conectamos por medio de cable ethernet nuestra PC y la tarjeta embebida RaspberryPI

IMPORTANTE: La RaspberryPI debe estar encendida.

Paso 2: Abrimos PuTTY

Paso 3: En host Name colocamos: raspberrypi.local ; en Port: 22

Paso 4: Click en Open

NOTA: Cuando nos conectamos por primera vez nos saldrá una advertencia, damos click en Accept

Paso 5: Colocamos el usuario y contraseña

usuario: pi

contraseña: raspberrypi

Ejemplo

login as: pi

pi@raspberrypi.local's password: raspberrypi

NOTA: La contraseña no se muestra al escribirla, solo debemos escribirla correctamente y dar enter.

Conexión con FileFilla

IMPORTANTE: Debemos tener la RaspberryPi encendida y correctamente conectada a la PC por medio del cable Ethernet

Paso 1: Abrimos FileFilla

Paso 2: Llenar con la siguiente información

Host: raspberry.local

Username: pi

Password: raspberry

Port: 22

Paso 3: Click en Quickconnect

Realizados los pasos correctamente tendremos acceso a los archivos contenidos en la Raspberrypi.
