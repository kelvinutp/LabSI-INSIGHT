¿Qué es la terminal?

La terminal es una interfaz de línea de comandos en sistemas operativos Linux. Es una forma de interactuar con el sistema operativo y realizar tareas, como navegar por los archivos, ejecutar programas y scripts, y realizar tareas de administración del sistema, sin la necesidad de una interfaz gráfica de usuario (GUI).

En lugar de clicar en iconos y menús para realizar tareas, en la terminal, los usuarios ingresan comandos a través de la línea de comandos y reciben una respuesta en texto plano. Esto puede ser más rápido y eficiente que una GUI en algunos casos, y también permite a los usuarios automatizar tareas mediante la creación de scripts.

En resumen, la terminal es una herramienta poderosa y esencial para los administradores de sistemas y los desarrolladores que trabajan con sistemas operativos Linux.

Comandos más usados 

ls: Lista los archivos y directorios en una ruta específica.
cd: Cambia a un directorio específico.
pwd: Muestra la ruta actual del directorio.
nano: Abre un editor de texto en la terminal para editar un archivo.
clear: Limpia la terminal

Navegando entre archivos

Puedes navegar entre directorios en Raspberry Pi usando el comando cd.

Sintaxis: cd [ruta_del_directorio]

Ejemplo:

Para ir al directorio raíz, usa: cd /.
Para ir al directorio anterior, usa: cd ...
Para ir a un directorio específico, por ejemplo Documents, usa: cd Documents.

Ejecutando scripts Python

sudo python3 <nombre_del_script.py> : Ejecuta un script de python
nano <nombre_del_script.py> : abre editor de texto en la terminal

Galería de comandos

mkdir: Crea un nuevo directorio.
rmdir: Elimina un directorio vacío.
cp: Copia un archivo o directorio a otra ubicación.
mv: Mueve o renombra un archivo o directorio.
rm: Elimina un archivo.
touch: Crea un nuevo archivo vacío.
