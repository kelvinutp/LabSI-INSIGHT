Modulo

Este módulo de cámara tiene una resolución de 5 megapixeles y cuenta con una librería de Python dedicada a su funcionamiento.
Esta cámara puede ser utilizada tanto para tomar fotos como para grabar videos.
La librería dedicada también cuenta con funciones para temas de configuración de la imagen capturada, como el balance de blancos, obturador, entre otros.
Resoluciones de video: 1080p a 30 fps, 720p a 60fps o 480p a 90fps.

Conexiones
